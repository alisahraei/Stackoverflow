Maybe you can use CSS. Something like:

#lightbox #imageContainer img {
    max-width: 600px;
}

/*********************************************
 *
 * NOTE: "max-width" is not supported by IE6.
 *       If al your images are bigger than
 *       the "default", just use "width".
 *
 *********************************************/
